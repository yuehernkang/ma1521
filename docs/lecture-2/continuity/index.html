Continuity

Let fff be a real-valued function defined on some interval lll (e.g. (aaa, bbb), or (aaa, bbb] or (aaa, ∞\infty∞)). Let ccc be a point in lll.

Continuity at a point​

Case 1 c is an interior point

  • fff is continuous at x=cx=cx=c if
    1. lim⁡x→cf(x)\lim\limits_{x \to c} f(x)x→clim​f(x) exists,
    2. lim⁡x→c−f(x)=f(c)\lim\limits_{x \to c^-} f(x) = f(c)x→c−lim​f(x)=f(c)

Case 2 c is the left hand end-point

  • fff is continuous at x=cx=cx=c if
    1. lim⁡x→c+f(x)\lim\limits_{x \to c^+} f(x)x→c+lim​f(x) exists,
    2. lim⁡x→c+f(x)=f(c)\lim\limits_{x \to c^+} f(x) = f(c)x→c+lim​f(x)=f(c)

Case 3 c is the right hand end-point

  • fff is continuous at x=cx=cx=c if
    1. lim⁡x→c−f(x)\lim\limits_{x \to c^-} f(x)x→c−lim​f(x) exists,
    2. lim⁡x→c−f(x)=f(c)\lim\limits_{x \to c^-} f(x) = f(c)x→c−lim​f(x)=f(c)

Continuity on an interval​

  • fff is continuous on an interval if fff is continuous at x=cx=cx=c for all points ccc in lll.

Results (Limit Laws)​

  • If fff and ggg are continuous at x=cx=cx=c, then for any constant kkk and any positive constant nnn, each of the following function is continuous at x=cx = cx=c.
  1. f+gf + gf+g
  2. fnf^nfn
  3. kfkfkf
  4. fgfgfg
  5. fg\frac{f}{g}gf​ provided g(c)≠0g(c) \not ={0}g(c)=0
  • If ggg is continuous at x=cx=cx=c and fff is continuous at x=g(c)x=g(c)x=g(c), then the composite function f∘gf\circ gf∘g is continuous at x=cx=cx=c
  • (Note (f∘g)(x)=f(g(x))(f \circ g)(x) = f(g(x))(f∘g)(x)=f(g(x)))

Example (Discontinuity)​

Find the points of discontinuity​

** If a graph is continuous, we can draw it without lifting our pencil
** It is discontinuous when we have to pick up our pencil
** There should not be any break or holes on the curve for it to be continuous

  • Points of discontinuity : x=2x=2x=2 and x=4x = 4x=4
